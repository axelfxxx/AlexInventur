AlexInventur – Dynamische Gerätedetails

Neu:
- Artikel-/Gerätedetails werden rechts neben dem Grid angezeigt.
- Basisdaten und CSV-Zusatzfelder werden dynamisch aus den importierten CustomFields gelesen.
- Technische Felder werden automatisch gruppiert:
  - System
  - Identifikation
  - BIOS
  - CPU
  - Arbeitsspeicher
  - Datenträger
  - Netzwerk
  - Sonstige Felder
- Detailpanel aktualisiert sich bei Grid-Auswahl, Suche, Filter und Zellbearbeitung.
- Bei sehr kleinen Fenstern wird das Detailpanel automatisch ausgeblendet, damit das Grid bedienbar bleibt.

Hinweis:
- Build wurde in dieser Umgebung nicht ausgeführt, da kein .NET SDK verfügbar ist.
